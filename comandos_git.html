git checkout -b maria: Vamos a dar de alta la rama llamada "maria".

git checkout main: Volvemos a la rama main.

git checkout maria: Volvemos a esta rama.

git add . ; git commit -m "mensaje": Pero tenemos que setear a donde apunta maria.

git push origin maria: Ahí sí va a subir los cambios.

git branch: Lista las ramas locales.

git revert + nro de commit; git status: Deshace un commit y muestra el estado actual.

git pull: Descarga todo.
